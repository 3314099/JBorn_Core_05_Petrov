1. -1 в byte : да, значение будет равно -1
2. -1 в char : да, но: В Java для char используется кодировка Unicode и для хранения Unicode-символов.
Диапазон допустимых значений - от 0 до 65536 (отрицательных значений не существует), таким образом
система расценит запись как арифметическую операцию и вычтет из максимально возможного значения(65536) единицу.
Результат 65535.
